Open Source

 

Awesome RubyMotion  

Een gecureerde lijst met links over de programmeeromgeving RubyMotion.

 

AWS SDK voor iOS & macOS  

De Software Development Kit voor Amazon Web Services is oorspronkelijk ontwikkeld voor iOS. Ik heb de SDK aangepast zodat deze ook te gebruiken is om macOS applicaties te ontwikkelen. De broncode van de voorbeeldapplicaties laten zien hoe het werkt.

 

PDF-Images  

Objective-C CLI app om afbeeldingen uit pdf-bestanden te extraheren.

 

Capistrano TYPO3  

Dit is een Capistrano plugin gemaakt om TYPO3 automatisch uit te rollen in een CD/CI workflow. Daarnaast is deze plugin voor capistrano ook als hulpmiddel te gebruiken om snel te kunnen werken met een TYPO3 omgeving in een Vagrant virtual machine.

 

Exact Online Client Library voor iOS en macOS  

Softwarebibliotheek om iOS and macOS te laten werken met de Exact Online API. Bekijk de broncode van de voorbeeldapplicaties om te zien hoe het werkt. Je zou hier bijvoorbeeld een Exact Online urenregistratie app voor macOS mee kunnen maken.

 

YAPDFKit  

Yet another PDF Kit. Onafhankelijke Objective-C-bibliotheek voor het uitlezen en manipuleren van pdf-documenten. Bruikbaar lesmateriaal om te begrijpen hoe een pdf-bestand werkt. YAPDFKit is de motor van PDF Letterhead.

 

Zabbix Agent Installer voor Mac OS X  

Programma om de Zabbix Agent op een macOS-server te installeren.

2004 - 2015

PAS3, drukwerkapp voor franchise bedrijven 2004 - 2015

PAS3 is een eenvoudige webapplicatie die Adobe InDesign aanstuurt om persoonlijke voorkeuren en eigen gegevens door te voeren in professioneel opgemaakt promotiemateriaal. Oorspronkelijk ontwikkeld voor alle Suzuki-dealers in Nederland. Later in gebruik genomen door diverse dealer- en franchiseorganisaties. Conceptueel is PAS3 de voorloper van Kaftmeister.

2001 - 2008

eGroupWare en JiNN CMS 2001 - 2008

In de beginjaren van Lingewoud was ik medeoprichter van het project dat de eGroupWare Office Suite ontwikkelde. De suite was een online omgeving waarin meerdere programma’s draaiden. 1999 - 2002

Linuxstart.nl 1999 - 2002
