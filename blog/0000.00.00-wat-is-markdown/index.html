Wat is Markdown?

technologie | schrijven | Markdown

Introductie van Markdown voor niet-technische schrijvers

21 mei 2019
technologie | schrijven | Markdown

Markdown is stilletjes de schrijfwereld aan het veroveren. Softwareontwikkelaars waren de eerste die de opmaaktaal omarmden, en nu volgt langzaam de rest van de wereld.

Waarom?
Omdat het eenvoudig te leren is en ontzettend handig als je eenmaal gewend bent te schrijven in Markdown. Ik zal later meer schrijven over de voordelen, maar in dit artikel leg ik uit wat Markdown is.

Markdown is een opmaaktaal uitgevonden door John Gruber, een invloedrijke techjournalist. Hij wilde de teksten die hij schreef voor z’n website eenvoudig voorzien van opmaak, maar zonder HTML-codes te gebruiken. Hij bedacht een aantal eenvoudige markeringen die aangeven hoe de tekst moet worden opgemaakt. Deze markeringen zijn zo logisch dat je zonder Markdown-kennis, de betekenis van de opmaak meestal direct begrijpt.

Een paar voorbeelden zodat je het idee meteen snapt.

Allereerst een vette kop. De dikste, namelijk voor de titel van je tekst. In HTML is dat een H1 en in Word kies je hiervoor de Kop 1 uit je stijlpaneel. In Markdown maak je deze titel door je woorden te onderstrepen met =tekens.

Ander voorbeeld. Een opsomming. Een logische manier om een opsomming te maken is per item een regel te starten met een -teken, dan een spatie en dan de tekst van het item. Het volgende item net zo op een nieuwe regel. Precies zo maak je een opsomming in Markdown.

Wat is het meest voor de hand liggend om een horizontale lijn weer te geven? Liggende streepjes. In Markdown minimaal drie, meer mag ook.

Normale tekst type je gewoon als normale tekst. Wil je een witregel, maak dan een witregel door een lege regel tussen twee alinea’s over te laten.

Kijk hoe eenvoudig en vanzelfsprekend deze paar voorbeelden eruit zien.

Waarom Markdown
===============

Wat Markdown is, is één ding. Maar waarom moet ik zo nodig Markdown gebruiken? Ik kan toch gewoon schrijven in Word. Dat ben ik gewend.

---

Markdown heeft talloze voordelen.

* Het is eenvoudig te leren
* Je hoeft er geen speciale software voor de installeren
* Je kunt Markdown-teksten naar HTML-bestanden omzetten

Dit stukje Markdown kan op allerlei manieren worden weergegeven. De opmaakvormgeving wordt later toegevoegd. Ook wordt later bepaald hoe of waar de tekst wordt weergegeven. De opgemaakte tekst kan onderdeel zijn van je website, of het kan een PDF-bestand zijn op je briefpapier.

Als je met Markdown je teksten schrijft heb je altijd 2 versies van een tekst. De bronversie en de naar het einddoel geconverteerde versie. De schrijver werkt in de bronversie. Hiervoor gebruik je een programma dat bestanden kan opslaan als platte tekst. Platte tekst is voor een computerprogramma de meest eenvoudige manier om tekstbestanden op te slaan. Op vrijwel elke computer staat wel een programma dat dit kan. Toch zijn er voor speciaal voor Markdown programma’s ontwikkeld.

De belangrijkste toegevoegde waarde van een Markdown-programma is dat direct de typografisch opgemaakte tekst wordt getoond. Net als je gewend bent van bijv. Word of Pages. Deze programma’s kunnen je Markdown-bronbestanden ook direct exporteren naar een geconverteerde versie. Meestal PDF of HTML.

Als je Markdown wilt uitproberen raad ik aan om met zo’n Markdown-programma te starten.Er is veel keuze in Markdown-programma’s en is er altijd wel een gratis te vinden. Voor macOS is er bijvoorbeeld het programma MacDown. Dit programma hanteert de veelgebruikte gesplitste weergave met links de platte tekst en rechts de opgemaakte versie van de tekst. Markdown Monster voor Windows gebruikt dezelfde geplitste weergave. Beide programma’s zijn gratis en open source.

Een Markdown-programma dat opvalt vanwege de gebruiksvriendelijkheid en elegantie is Typora. Beschikbaar voor alle platformen. Dit programma is terwijl ik dit schrijf nog als beta gratis te downloaden. Het lijkt er echter op dat de makers Typora klaarstomen om als een betaald programma te worden verkocht. Mogelijk zelfs met een abonnementsmodel.

Typora gedraagt zich eigenlijk een beetje als een klassieke tekstverwerker. Je ziet direct de typografisch opgemaakte tekst. Wil je tijdelijk meer controle, dan kun je kiezen om de tekst te tonen als Markdown-broncode. Wie kent nog het onderwaterscherm van WP 5.1?

Je kunt tijdens het typen direct alle Markdown-markeringen kunt intypen. Bij een normale tekstverwerker zou je dan zoeken in het menu, of snelkoppelingen gebruiken voor bijvoorbeeld vet, of onderstreept. Deze mogelijkheden biedt Typora ook. Je kunt linksom rechtsom of rechtdoor.

De weergave in het Markdown-programma Typora is tijdens het schrijven prachtig opgemaakt in een thema naar keuze. Je kunt ook nieuwe thema’s downloaden.

Ik heb geen Typora-aandelen, maar ik raad je zeer aan dit programma eens te proberen. Er is trouwens een open source concurrent van Typora, Mark Text. Dit programma is ontwikkeld met dezelfde visie als Typora. Elegant en gebruiksvriendelijk en Mark Text draait ook op alle Windows, Mac en Linux. Zeker ook de het uitproberen waard.

Schrijf je veel, of ben je veel met internet of webdevelopment bezig, vroeg of laat zul je met Markdown in aanraking komen. Ga er alvast een mee spelen. Het is echt leuk.
